Simple app to setup server config files and start the server with set configuration.


## Assumptions:
- This app requires that you have SteamCMD installed. To install SteamCMD follow the guide [here](https://developer.valvesoftware.com/wiki/SteamCMD)
- Secondly this app assumes that you installed SteamCMD here: ```C:\SteamCMD```
